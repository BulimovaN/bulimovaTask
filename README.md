# Bulimova Task

REST API для управления пользователями и задачами.

Проект выполнен на Spring Boot с использованием JWT и PostgreSQL.

---

## Технологии
- Java 22
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- PostgreSQL
- Docker Compose

---
